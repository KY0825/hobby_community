# 要件定義
### ログイン機能
目的
ユーザーのアカウント作成のため

詳細
gemのdeviseを用いて実装する。

ユーズケース
ルームの作成、チャットの作成をするためには必須

### マイページ(ユーザー詳細ページ）
目的
自身の登録情報の確認、編集
ルーム、チャットの管理のため

（友達機能実装後）
まだ友達になっていない他のユーザーに友達申請を送ることができる。

詳細
詳細ページ用のviewを作成して実装する。

ユーズケース
・自身の登録情報の確認、編集をする。
・ルーム、チャットの管理をする。

### ジャンル・カテゴリー追加申請フォーム
目的
ユーザーがトークルームの設定に必要なジャンル、カテゴリーを新規で追加申請をするため

（友達機能実装後）
まだ友達になっていない他のユーザーに友達申請を送ることができる。

詳細
外部の無料のフォームサービスを用いて実装する。

ユーズケース
・ユーザーが自身のルームを作成した際に、希望するカテゴリー、ジャンルがない場合に申請をする。

許可されれば追加され、ユーザーには別途連絡する。

### 参加グループ一覧機能
目的
ユーザーが自身が参加しているルームの一覧を確認するため

詳細
eachメソッドで表示し、上から３つを表示する。

それ以降は「さらに表示」を押すことで、表示されるようにする。

ユーズケース
・ユーザーは自身が参加しているルームの一覧を確認することができる。

### チャットルーム作成機能
目的
ユーザー同士でチャットを行うため。

詳細
userとroomの間に中間テーブル「user_room」を作成して実装する。

ユーズケース
・ユーザー同士で特定のカテゴリ、ジャンルが選択され、作成されたルーム内でチャットを行うことができる。

・ユーザーはチャットの新規投稿、編集、削除が可能である。

・ルームオーナーに管理者権限を付与する。（詳細は今後詰める）

### グループ検索機能
目的
ユーザーがトークルームの検索を行うため。

詳細
searchメソッドを用いて実装する。

ユーズケース
・ユーザーは参加したいグループをジャンル、カテゴリで検索が可能になる。


### 友達申請機能
目的
ユーザー間で友達申請を行うため。

詳細
導入方法検討中

ユーズケース
・ユーザー間で友達になることができる。

### 友達検索機能
目的
各ユーザーが登録している、ジャンル、カテゴリーを選び、友達申請を送ることができる。

詳細
searchメソッドで実装予定

ユーズケース
・ユーザーはジャンル、カテゴリで検索して、友達になりたいユーザーを検索できる。


### フレンドチャット機能
目的
管理人の友達同士でチャットが可能になる。

詳細

ユーズケース
・管理人の友達同士でチャットが可能になる。
（チャット参加者の友達、かつ、管理人と友達ではないユーザー）の参加については今後検討する。

# データベース設計

## users テーブル

| Column             | Type   | Options     |
| ------------------ | ------ | ----------- |
| nickname           | string | null: false |
| email              | string | null: false |
| encrypted_password | string | null: false |
| last_name          | string | null: false |
| first_name         | string | null: false |
| last_name_kana     | string | null: false |
| first_name_kana    | string | null: false |
| birthday           | date   | null: false |
| category           | 未定    |             |
| genre              | 未定    |             |

### Association

- has_many :groups
- has_many :messages


## groups テーブル

| Column             | Type       | Options     |
| ------------------ | ---------- | ----------- |
| name               | string     | null: false |
| text               | text       | null: false |
| category_id        | integer    | null: false |
| genre_id           | integer    | null: false |

### Association

- has_many :users
- has_many :messages


## users_groups テーブル

| Column | Type       | Options                        |
| -------| ---------- | ------------------------------ |
| user   | references | null: false, foreign_key: true |
| group  | references | null: false, foreign_key: true |

### Association

- belongs_to :users
- belongs_to :messages


## messages テーブル

| Column | Type       | Options                        |
| ------ | ---------- | ------------------------------ |
| text   | text       | null: false                    |
| user   | references | null: false, foreign_key: true |
| group  | references | null: false, foreign_key: true |

### Association
- belongs_to :users
- belongs_to :messages
